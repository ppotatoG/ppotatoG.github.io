### 제출한 답

유클리드 호제법이 아닌 다른 방법으로 풀면, 런타임에러가 난다ㅜㅜ
```js
function solution(a, b) {
    const gcd = (a, b) => a % b === 0 ? b : gcd(b, a % b);
    return [gcd(a, b), a * b / gcd(a, b)];
}
```
