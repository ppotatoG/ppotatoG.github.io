### 제출한 답

값이 0 인 변수 `tmp`

중복을 돌며 `tmp`에 값을 담아주기

`단, 금액이 부족하지 않으면 0을 return 하세요.`라는 제한이 있으니 조건부로 값 리턴하기

`Math.abs`는 왜 쓴거지 ?.?
```js
function solution(p, m ,c){
    let tmp = 0;

    for(let i = 1; i < c + 1; i++){
        tmp = tmp + p*i;
    }

    return tmp <= m ? 0 : Math.abs(tmp) - m;
}
```
