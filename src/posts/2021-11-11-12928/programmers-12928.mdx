### 제출한 답

`n`의 약수를 `answer`에 더해주기
```js
function solution(n) {
    let answer = 0;
    for(let i = 1; i <= n; i++){
        if(n % i == 0) answer +=i;
    }
    return answer;
}
console.log(solution12928(12)) // 28
console.log(solution12928(5)) // 6
```
