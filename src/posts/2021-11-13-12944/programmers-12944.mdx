### 제출한 답

[reduce](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/Reduce)로 모든 값을 더한 후 `num`의 길이로 나눠주기
```js
function solution(num){
    return num.reduce((a, b) => a + b)/num.length;
}
```
